=====================================================
VERCEL ENVIRONMENT VARIABLES SETUP
=====================================================

YOUR BFF URL: https://bff-donganksa.vercel.app

=====================================================
FRONTEND ENVIRONMENT VARIABLES
=====================================================
Go to: https://vercel.com → Your Frontend Project → Settings → Environment Variables

Add these variables:

1. VITE_BFF_URL
   Value: https://bff-donganksa.vercel.app

2. VITE_API_URL
   Value: https://bff-donganksa.vercel.app/api

3. VITE_API_BASE_URL
   Value: https://bff-donganksa.vercel.app/api

Then click "Redeploy" in the Deployments tab.

=====================================================
BFF (BACKEND) ENVIRONMENT VARIABLES
=====================================================
Go to: https://vercel.com → Your BFF Project → Settings → Environment Variables

Add these variables:

1. DATABASE_URL
   Value: your-postgresql-connection-string

2. JWT_SECRET
   Value: (generate with: node -e "console.log(require('crypto').randomBytes(64).toString('hex'))")

3. SERVICE_TOKEN
   Value: (generate with: node -e "console.log(require('crypto').randomBytes(64).toString('hex'))")

4. FRONTEND_URL
   Value: https://grc-platform-ps9oei4eo-donganksa.vercel.app

5. NODE_ENV
   Value: production

6. PORT
   Value: 3005

Optional (if using integrations):
7. MICROSOFT_CLIENT_ID
8. MICROSOFT_CLIENT_SECRET
9. MICROSOFT_TENANT_ID
10. STRIPE_SECRET_KEY
11. STRIPE_PUBLISHABLE_KEY
12. ZAKAT_IE_API_KEY

Then click "Redeploy" in the Deployments tab.

=====================================================
QUICK STEPS
=====================================================
1. Add VITE_BFF_URL to frontend in Vercel dashboard
2. Redeploy frontend
3. Add DATABASE_URL and other vars to BFF
4. Redeploy BFF
5. Test connection!
